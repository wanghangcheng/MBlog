# MBlog
MBlog博客项目

##### 首先解压压缩包

##### 其他文件中包含了头像文件和sql文件

##### sql文件夹里的sql文件，现在mysql里创建test数据库，然后执行sql文件

##### 在F盘下在创建WEB文件夹，用于存放头像文件和日志文件

##### 如果没有F盘，可以先放在任意盘符下，然后将application.properties文件中的路径的F改为你放的盘符

## 说明

##### 项目使用SpringBoot+freemaker+Ehcache+MyBatis+SpringBoot DATAJPA
##### 包含了前显示和后台管理，基本功能全部实现，部署后可以加上第三方登录，已经预留了，直接换上api接口就好
##### 邮件发送可以在工具类中换上自己的qq邮箱和stmp授权码
##### 浏览量上使用了Spring的定时任务实现
##### 页面布局方面使用了BootStrap的栅格系统，前后台都支持移动端显示
